# MiniGPT

A minimalistic ChatGPT UI. Try it out here ([minigpt.pages.dev](https://minigpt.pages.dev))

## Features

- Your API key is stored locally
- Your conversations are stored locally
- Minimalistic UI
- PWA support

## Tech Stack

- NextJS
- Tailwind CSS

## Run Locally

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
